# Invoice Constructor

Browser-based invoice generator with real-time preview and PDF export. Built as a personal project to learn TypeScript.

![TypeScript](https://img.shields.io/badge/TypeScript-blue?style=for-the-badge) ![React](https://img.shields.io/badge/React-navy?style=for-the-badge) ![Vite](https://img.shields.io/badge/Vite-purple?style=for-the-badge)

An example invoice generated by this app can be found [here.](https://github.com/notred27/Invoice-Constructor/blob/main/README/example-invoice.pdf)

![Browser view of the Invoice Constructor](README/view.png)

## Features

- Set sender and recipient information
- Add invoice metadata
  - ID
  - issue date
  - due date
- Add a variable number of line items with descriptions and amounts
- Automatically calculated running totals
- Real-time preview and one-click PDF download from the browser

## How to Run

Requires Node.js and npm.

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
